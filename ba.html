Business Analytics Practicals

Practical 1: Import Legacy Data

  1. Open Power BI software.
  2. Go to 'Get Data' option under External Data group.
  3. Select the required file format, enter the file name, and click open.
  4. A navigator window opens. Select the sheets you want to work on and click 'Edit'.
  5. Power Query editor opens. Go to Get Data and select OData, then enter the URL.
  6. Select the data sheets you want and click 'Load'.
  7. Click on the Model View icon to view relationships.

Practical 2: ETL Process in Power BI

  1. Load data from OData Feed.
  2. Open Power BI and click on 'Get Data', then select OData Feed.
  3. Paste the URL: 'https://services.odata.org/V3/Northwind/Northwind.svc/'.
  4. Select the Products table, click on 'Transform Data'.
  5. Remove unnecessary columns and change data type of the UnitsInStock column to whole number.
  6. Save and close the query editor. Power BI will load the data.
  7. Check all fields and view the data representation in chart form.

Practical 4: Data Analysis in Excel

  1. Open Excel and search for 'From Data Connection Wizard'.
  2. Select Microsoft SQL and connect to your database.
  3. Select Sales_DW (already created in SQL), check all dimensions and import relationships.
  4. Create a Pivot Table and a Pivot Chart to analyze the data.
  5. Drag and drop fields to filter and analyze the data as needed.

Practical 5: What-If Analysis

  1. Open Excel and save your file with a relevant name.
  2. Insert your data in a table format.
  3. Use the formula =SUM(C3:C6) to calculate the sum of prices.
  4. Use =C8*10 to calculate the price of 10 systems.
  5. Go to the Data tab, select 'What-If Analysis', and choose 'Scenario Manager'.
  6. Add new scenarios and input the values for different scenarios.
  7. Click 'Show' to apply the scenario changes.

Practical 6: Linear Regression

  1. Use the `lm()` function to create a relationship model between predictor and response variables.
  2. Syntax: `lm(formula, data)` where `formula` represents the relation and `data` is the vector to apply it.
  3. Create a relationship model and use `print(relation)` to display it.
  4. Get the summary using `summary(relation)`.
  5. Predict a new value using the `predict()` function.
  6. Visualize the regression using the `plot()` function to create a graph of the model.
